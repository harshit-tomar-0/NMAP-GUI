🛠️ Nmap GUI Scanner (Python)
📌 Description

A GUI-based network scanning tool built using Python that integrates Nmap to perform port scanning and network reconnaissance in a user-friendly interface.

🚀 Features
<ol>Scan target IP/domain</ol>
<ol>Port scanning using Nmap</ol>
<ol>Simple GUI interface</ol>
<ol>Displays scan results</ol>
⚙️ Tech Stack
<ol>Python</ol>
<ol>Nmap</ol>
<ol>Tkinter</ol>
▶️ How to Run
<ol>pip install python-nmap</ol>
<ol>python main.py</ol>
